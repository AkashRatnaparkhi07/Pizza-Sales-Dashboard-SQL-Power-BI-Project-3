# Pizza-Sales-Dashboard-SQL-Power-BI-Project-3
The Pizza Sales Dashboard is designed to help stakeholders monitor key performance indicators (KPIs) and identify growth opportunities. The project involves extracting data from a SQL database, performing complex transformations, and building an interactive dashboard to visualize sales dynamics.
📊 Key Performance Indicators (KPIs)
The dashboard successfully calculates and tracks the requested core metrics:

Total Revenue: The sum of the total price of all pizza orders ($817.9K).

Average Order Value: The average amount spent per order ($38.31).

Total Pizzas Sold: The sum of the quantities of all pizzas sold (49,574).

Total Orders: The total number of orders placed (21,350).

Average Pizzas Per Order: The average number of pizzas sold per order (2.32).

📈 Visualizations & Business Insights
Based on the chart requirements, the report translates raw data into actionable insights through targeted visualizations:

Order Trends (Daily & Monthly): Bar and line charts identifying fluctuations in order volume. The data reveals that orders peak on weekends (Friday/Saturday evenings) and during the months of July and January.

Sales by Pizza Category: A donut chart showing the distribution of sales, highlighting that the Classic category contributes to the maximum sales and total orders.

Sales by Pizza Size: A donut chart illustrating that Large size pizzas are the primary driver of overall sales.

Total Pizzas Sold by Category: A bar visualization ranking categories by overall volume to help manage inventory.

🛠️ Technology Stack
Power BI: Utilized for data modeling, creating custom DAX measures, and designing an intuitive, client-ready user interface.

SQL: Used for initial data extraction, data validation, and structuring queries to match the required KPIs.
